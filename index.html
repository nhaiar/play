Open Developer Tools > then under Images, click the Image, then type $0.naturalWidth to view the width of the image. To find the number of bits, multiply the Pixels by the Bits per Pixel. Less pixels x better compression = less bytes. Always use max-width: 100%; for the property of images in css. There is also a calc tool within the browser devtools. For example, width: calc((100% - 10px)/2);
